  • Фронтендом я интересовался еще в школе. В 9 классе на уроках информатики одним из заданий было написание собственного сайта.
    К сожалению те файлы не сохранились, но тогда ничего сложного я не писал, просто несколько html страничек, связанных между собой ссылками,
    никакого css там не использовалось. Однако после этого мне стало интересно, как сайты работают изнутри.
    Это и было одной из причин, по которой я выбрал изучать курс фронтенда.

  • Другой причиной является то, что этот курс является практическим, то есть тут можно получить много навыков, которые затем можно использовать.
    Это мне нравится куда больше, чем учить большое количество теории, без возможности применить её на практике.
    К тому же практические знания редко бывают лишними, ведь никогда не знаешь когда они могут пригодится

  • Прежде всего я надеюсь, что к концу курса научусь писать сайты так, чтобы на них можно было нормально смотреть. То есть хочется научиться
    использовать возможности html и css для создания приятного для глаз дизайна. Также интересно узнать, чем отличается работа веб-программистов
    от обычных программистов. Возможно веб-программирование понравится мне даже больше, чем обычные языки программирования.
